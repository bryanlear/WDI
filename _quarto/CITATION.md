# Citation

```verbatim
To cite package ‘WDI’ in publications use:

  Arel-Bundock V (2025). _WDI: World Development Indicators and Other
  World Bank Data_. R package version 2.7.9,
  <https://vincentarelbundock.github.io/WDI/>.

A BibTeX entry for LaTeX users is

  @Manual{,
    title = {WDI: World Development Indicators and Other World Bank Data},
    author = {Vincent Arel-Bundock},
    year = {2025},
    note = {R package version 2.7.9},
    url = {https://vincentarelbundock.github.io/WDI/},
  }
```
